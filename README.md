# Gymnastics Training & Membership Management System

## Overview
A web-based system developed to manage gymnastics club members and their information. The system allows users to register members, view records, update details, delete records, and generate reports.

## Features
- Register new members
- View member records
- Update member information
- Delete member records
- Generate reports
- Confirmation page
- Dashboard

## Technologies Used
- PHP
- MySQL
- HTML
- CSS

## Installation
1. Import `gymnastics_system.sql` into MySQL.
2. Place the project files in your web server directory.
3. Start Apache and MySQL.
4. Open the project in your browser.
